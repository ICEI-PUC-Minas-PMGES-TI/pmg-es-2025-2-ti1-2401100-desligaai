# Portal de Autenticação (HTML + CSS + JavaScript)

Projeto simples que simula cadastro e login usando apenas arquivos estáticos.

## Como usar

1. Abra o arquivo `index.html` diretamente no navegador (duplo clique ou arraste para a janela do navegador).
2. Cadastre um usuário preenchendo nome, Gmail e senha.
3. Aguarde a mensagem de sucesso e volte para a tela de login automaticamente.
4. Faça login com o usuário criado e visualize os dados salvos.
5. Use o botão de logout para voltar à tela inicial.

Todos os dados ficam somente na memória enquanto a página estiver aberta. Não há servidor nem dependências externas além do Bootstrap CDN.
